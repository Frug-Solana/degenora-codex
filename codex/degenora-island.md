# Degenora Island

Degenora Island is the narrative setting for FRUG.

Think of it like an open world map.

## What lives here
- Territories
- Creatures
- Field reports
- Specimens (pump.fun native projects that are documented inside the world)

## Terminology
- **Terminal 7-B**: an in progress public interface concept, a terminal style codex for browsing the archive
- **Specimen**: a documented project entry with an identity, placement, and lore hooks
